2048 game written on Scala. cs109ui.jar (author: Otfried Cheong) file should be moved to lib folder (C:\Program Files\scala-2.11.7\lib)
Then 2048object.scala file should be compiled (ex: fsc 2048object.scala) and run (ex: scala play2048).
This project is based on practice problem http://otfried.org/courses/cs109/project-2048.html
